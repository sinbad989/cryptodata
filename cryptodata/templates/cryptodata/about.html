Cryptocurrency Specs for Intelligent Investing.

It is beneficial to keep a collection of information about the cryptocurrencies in your portfolio or the assets you might want to keep an eye out for. This will help you be an intelligent investor. For each coin in the market, a short specs will give you an overview of the true nature of the coin and will gave you an idea if you've made a sound investment on a coin. This weeds out the coins which are likely to be a scam.
The data here are from various sources and is collated to provide an easier overview.

This is a collection of important coin specs in a single site. The data are from various sources:

  • Rank:
    • ranking based on the value of market capitalization
  • Symbols:
    • icons are from coinranking, they provide a beautiful icons.
  • Name (type):
    • contains the link of the crypto and its respective type, either a coin or a toke.
  • Price:
    • base on coinmarketcap data API, you can get an updated price by refreshing the page.
  • Marketcap:
    • extracted using coinmarketcap API
  • Avail. Supply (Max Supply):
    • comes from various sources in the net, including coinmarketcap
  • Community score:
    • an average score from developer score,social media interest, and search engine based scoring, data comes from coingecko.
  • Block Time:
    • It is a measure of how long it will take the hashing power of the network to find a solution to the block hash.
